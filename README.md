# 🎬 Movie Recommender System

An interactive Movie Recommendation System built using Machine Learning and Streamlit that recommends similar movies based on content similarity.

---

## 🚀 Features

- 🎥 Recommend similar movies instantly
- ⚡ Fast and interactive Streamlit web app
- 🧠 Content-based recommendation system
- 📊 Similarity score powered recommendations
- 🎨 Clean and modern UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-learn
- Pickle

---

---

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/your-username/Movie-Recommender-System.git
```

Go to project directory:

```bash
cd Movie-Recommender-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```

---

## 🧠 How It Works

The recommendation engine uses content-based filtering and similarity scores to recommend movies similar to the selected movie.
